1.2.0

AMELIORATIONS

  • Les contenus récemment joués sont affichés dans la section correspondante de l'onglet Vidéos.
  • Marquez les contenus que vous souhaitez regarder plus tard. Vous les retrouverez dans la section correspondante de l'onglet Vidéos.
  • La lecture enchaîne les contenus automatiquement. Cette fonctionnalité peut être désactivée dans les réglages de l'application.
  • Ajout d'un onglet Profil qui vous permet vous connecter à votre compte maRTS, de gérer les données de l'application ainsi que ses réglages.
  • En vous connectant à un compte maRTS, vous pouvez automatiquement synchroniser vos émissions favorites, les contenus à consulter plus tard ainsi que les contenus regardés récemment entre vos appareils.
