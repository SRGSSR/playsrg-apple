3.0.0

Dear users,

We are happy to release version 3.0 of Play SWI, which contains several usability and feature improvements. We strive to continuously improve our products to deliver the best possible user experience.

We hope you will enjoy this new version and future updates. Here are the changes made in this version:

BOTTOM NAVIGATION

Tab-based navigation provides for easier navigation throughout the content:

  • Videos and TV shows are located under the ‘Videos’ tab.
  • The search is accessible at any time through a dedicated ‘Search’ tab.
  • Personalized features (history, favorites, notifications and downloads) and application settings are available from the ‘Profile’ tab.

IMPROVEMENTS

A few improvements have been made:

  • Subtitle selection is more intuitive and better takes into account device settings.
  • The mini player can be dismissed when not needed.
  • Long-pressing items on devices running iOS 13 displays a preview and available actions.
  • Presentation has been improved to better make use of the available screen space, no matter the chosen font size.
  • Chromecast can be directly enabled with the dedicated button at the top of the videos, audios and livestreams homepages.

FIXES

The following issues reported by our users have been fixed:

  • The subtitle choice is correctly preserved when enabling AirPlay.
  • Playback is not paused anymore when using Handoff. This change fixes an issue affecting users having a Mac (running macOS Catalina) and iOS devices registered with the same Apple ID, and for which using Safari on macOS would trigger unexpected playback interruptions on iOS via Handoff.
